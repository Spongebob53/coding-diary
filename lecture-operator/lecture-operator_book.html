Operator-21.11.11.


            이항 연산자(78p)
            
                - binary operator (operand가 둘인 연산자)
            
                - 소괄호를 묶어주면 별도 구분되어 우선 연산한다
            
                - 우선순위별 목록(결합방향,우선순위 수학 기본)
            
                    []
            
                    expre++, expre--			        ==> postfix 연산자(반대는 prefix 연산자)
            
                                                            증가 및 감소 시점 차이, 1을 더하거나 뺀 값을 대입
            
                    ++expre, --expre, +expre, -expre, ~, !	==> ~ : 모든 비트 반전, ! : not
            
                    *, /, %					            ==> 두 피연산자를 해당 기호대로 연산
            
                                                            % : 나눈 나머지 반환
            
                    +, -					            ==> 두 피연산자를 해당 기호대로 연산
            
                    <<, >>, >>>				            ==> bit shift 연산자(숫자만큼 비트열 이동)
            
                                                            >>는 음수일 경우 1로 채움
            
                    <, >, <=, >=, instanceof	        ==> 관계연산자(피연산자의 값을 비교)
            
                    ==, !=					            ==> 위와 더불어 true o false 반환
            
                    &	┐				                ==> 피연산자의 비트가 둘 다 1일 경우에 1반환
            
                    ^	├ 비트연산은 정수만 지원         ==> 피연산자의 비트가 다를 때 1반환
            
                    |	┘				                ==> 피연산자의 비트 중 1이 있으면 1반환
            
                    &&		논리연산자		            ==> and
            
                    ||	(피연산자는 true o false)		==> or
            
                    (조건)? expr1 : expr2			    ==> 삼항연산자
            
                                                            조건 충족시 expr1, 불충족 expr2 반환
                            
                                                            SCE(Short-Circuit-Evaluation) 주의
            
                    =, +=, -=, *=, /=, %=, &=, ^=, |=, <<=, >>=, >>>=
            
                                                        ==> 복합 대입연산자는 형변환이 필요 없음
            
                                                            a = a + 1 ==> a += 1
