Conditional-21.11.12.


          if else(104p)
          
              - 구조
          
                  if(조건){						==> 조건은 true o false 반환
          
                      조건 true일 때 실행되는 영역
          
                  }else if(조건){
          
                      첫 번째 조건이 false이고
          
                      두 번째 조건이 true일 때 실행되는 영역
          
                  }else{
          
                    선행의 조건들이 모두 false일 때 실행되는 영역
          
                  }
          
              - tips
          
                - if, if절, else절에 속한 문장이 모두 하나인 경우 중괄호 생략가능
          
                - else if는 사실 if else를 중첩시킨 결과이다
          
          
          switch 와 block(112p)
          
              - else if 가 많다면 switch가 더 좋은 선택이 될 수도
          
              - 구조
          
                  switch(n){
          
                  case1:					  ==> case, defualt : Label 붙이기(위치 표시 목적)
          
                  case2:						     때문에 들여쓰기 하지않음
          
                      n이 2이면 여기서부터 실행	==> 이전의 case는 무시
          
                      break;				==> break : switch문을 벗어나겠다
          
                  case3: 
          
                      n이 3이면 여기서부터 실행
          
                  default:
          
                      해당 케이스가 없으면 여기서부터 실행
          
                  }
