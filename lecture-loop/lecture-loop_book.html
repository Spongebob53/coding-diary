          for / while / do while(119p)
          
              - 반복적 실행(앞선 if 와 switch는 선택적 실행)
          
              - 구조
          
                  - while(실행전 검사)
          
                      반복변수
          
                      while(반복조건){
                        
                          반복영역
          
                        반복변수 증감식
          
                      }
          
                  - do while(실행 후 검사)
          
                      do{
          
                        반복영역
          
                        반복변수 증감식
          
                      }while(반복조건)
          
                  - for 
          
                      for(반복변수; 반복조건; 반복변수 증감식){
          
                        반복영역
                        
                      }
          
          
          break continue
          
              - break : 반복문 빠져나가기
          
              - continue : 반복영역에서 남은 코드 무시하고 다음 반복조건 확인으로 넘어가기
          
              - 다중 for문 일 때 break tip
          
                  - Label 이용(들여쓰기 상관없이 잘보이는 위치에)
          
                      outer:	for(;;){
          
                                  for(;;){
          
                                      반복영역
          
                                      break outer;		==> 명시된 반복문까지 빠져나감
          
                                  }
                                  
                              }
